A Software Tester ensures that applications work as expected by identifying bugs and verifying functionality through both manual and automated testing. They play a key role in delivering quality products.

Key Responsibilities

  • Write and execute test cases and test plans
  • Perform manual, regression, and performance testing
  • Identify, document, and track bugs using issue tracking tools
  • Collaborate with developers to resolve defects
  • Ensure product quality through UAT (User Acceptance Testing)

Skills Required

  • Manual Testing Techniques
  • Automation Tools: Selenium, JUnit, TestNG
  • Bug Tracking Tools: JIRA, Bugzilla
  • Basic SQL for database testing
  • API Testing using Postman or REST-assured
  • Good understanding of SDLC & STLC
